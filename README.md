# VortexOfAgony

You dont need to use nodejs or go. Just use vortex of agony.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'vortex_of_agony'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install vortex_of_agony
